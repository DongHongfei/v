Ionic Framework 操作说明书

如果你用了 Ionic Framework 一段时间之后你会发现有好多经常重复用到的命令行语句。有时候经常会突然用到一个特殊的标记或者选项，所以我做了个小抄放在手边。这样在开发我的 app 的时候就能把注意力集中在编程上，不用时不时的去搜索一个特殊的命令了。
